About SQL

SQL, or Structural Query Language, is a standard language used to communicate with a relational database to store, retrieve and manipulate data.

A relational database is configured to identify relationships between data items, which are usually stored in tables.

Keep reading to learn more about SQL syntax.

CREATE command

The CREATE command in SQL creates a table in a database. The command structure is as as follows:

CREATE TABLE table_name ( column_name1 data_type, column_name2 data_type, ... );

Example

The following snippet of SQL creates a table of students in a database:

CREATE TABLE Student ( Student_ID VCHAR(10), Student_Fname VCHAR(20), Student_LName VCHAR(20), Student_Email VCHAR(20), PRIMARY KEY (Student_ID) );

In this example:

  • VCHAR is a variable character string data type.
  • The number in brackets after VCHAR (e.g. VCHAR(10) limits the number of characters in the string to a maximum (e.g. 10).
    • The PRIMARY KEY can be used to uniquely identify a row or record in the table. Here, Student_ID is the primary key; its value can uniquely identify a student record.

INSERT command

The INSERT command inserts a record into a table.

The command structure is as follows:

INSERT INTO table_name VALUES (value1, value2, ...);

Example

The following SQL command inserts a student record into the table "Student":

INSERT INTO Student VALUES (     '2017031001', 'James', 'Brown', 'jbrown@gmail.com' );

where character string data should be quoted in the command.

SELECT command

The SELECT command selects a record from a table.

The simplest command structure is as follows:

SELECT column1, column2, ... FROM table_name;

Example

To select a record from "Students" table:

SELECT ID, name, address, ... FROM Students;

SELECT all

To select all the columns available in a table, we would use an asterisk as per the following syntax:

SELECT * from table_name*

Example

To select all columns in our "Students" table:

SELECT * FROM students;

SELECT specific

To select specific columns available in a table, we would use the WHERE clause per the following syntax:

SELECT column1, column2, ... FROM table_name WHERE condition;

Example

To select records with first name "John" from our "Students" column:

SELECT * FROM Student WHERE Student_FName = 'John';

SQLite

SQLite is a database engine that can be used for:

  1. setting up a table
  2. saving the data from a form to that table
  3. retrieving the data from the table

SQLite is a self-contained, serverless, zero-configuration and transactional engine.

Unlike most other SQL databases, SQLite does not have a separate server process but rather reads and writes directly to ordinary disk files. A complete SQL database that includes multiple tables, indices, triggers and view is contained in a single disk file.
